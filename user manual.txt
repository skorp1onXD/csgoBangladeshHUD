How To USE : 

> Download and install node.js
> Copy everything from cfg to Counter-Strike Global Offensive\csgo\cfg
> Open RUN.bat as admin
> Open this link http://192.168.0.106:2626 in a browser 
> Go to create match select team then press save & force refresh HUD 
> Go to match tab copy the link from browser 
> Open OBS and create a browser source using the link 
> Ingame run game at fullscreen 1920x1080 
> Use exec observer command to remove base hud 
> Use exec observer_off to unbind all oberserver commands
